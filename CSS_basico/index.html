2. Tipos de Seletores

Selecione todas as tags <p> lá no arquivo style.css e perceba o que acontece.

Selectores de Classe (Identificados por ponto '.')

Eu sou um parágrafo dentro de uma 'div' com a classe "destaque".

A classe é como um clube VIP: qualquer elemento que receba a "pulseiraVIP" (nome da classe) receberá o mesmo estilo.

Seletor de ID (Identificador com hashtag '#')

Eu tenho um "ID" exclusivo marcado como 'paragrafo-unico'. O ID é como o CPF. Diferente da classe, não existem duas pessoas na mesma página carregando o mesmo CPF!
